# CLAUDE.md

Für dieses Repository gelten vollständig die Regeln aus [AGENTS.md](./AGENTS.md).

Vor jeder Änderung:

1. `AGENTS.md` lesen.
2. Passendes GitHub Issue prüfen oder anlegen.
3. Eigenen Issue-Branch verwenden.
4. Tests zusammen mit der Umsetzung ergänzen.
5. Erst bei grüner CI mergen.

Keine direkten Änderungen auf `main` und keine Umsetzung ohne vorheriges Issue.
