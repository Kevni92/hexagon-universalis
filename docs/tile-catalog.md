# Tile-Katalog

`src/data/tileCatalog.ts` trennt primären Oberflächentyp und
Relief-/Küsten-/Schneemodifikatoren. Die Zuordnung verwendet ausschließlich
aggregierte Erdwerte aus Terrain, Wasser, Landanteil, Küstenstatus und Höhe.
Die Katalogprofile sind eigene Low-Poly-Darstellungsprofile; sie enthalten
keine fremden Assets und erzeugen keine Geographie.
